### Prerequisites ###
In order to do any serious PureDarwin development, please familiarize yourself with the [DarwinBuild](Using%20DarwinBuild/DarwinBuild.md) project and the [MacPorts](/archive/developers/Using%20MacPorts/Using-MacPorts.md) project, as these are the key components of building software for Darwin.
### PureDarwin releases ###
See [PureDarwin Xmas](/archive/users/Xmas.md).
If you want to have a look at a minimalistic bootable Darwin 9 system, check out [PureDarwin nano](/archive/users/PureDarwin-Nano.md).
### Testers wanted ###
To qualify as a tester for the current PureDarwin installer, you'll need

-   An Intel Mac with Leopard (to run the installer and to test whether the PureDarwin USB drive can be booted)
-   A USB drive, 1GB or larger (will be completely erased)
-   Latest VMware or [VirtualBox](VirtualBox.md) release
-   Fast Internet connection (to download Darwin 9 packages)
-   A good backup since there are absolutely no guarantees

Please contact us in [irc #puredarwin](https://irc.freenode.net) if you want to be a tester.

### Useful tools ###
A utility for browsing through the contents of [binary roots](http://xref.puredarwin.org/) is available. It currently covers Darwin build 9F33, and provides links to relevant source code, and binaries.
